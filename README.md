# Install

Clone Repository:
<pre>
git clone https://github.com/valentinschaeffer/tic_tac_tou.git
</pre>
> Compile file: (in file directory)
> <pre>
> g++ -std=c++11 tiktactou.cpp -o tiktactou.out
> </pre>
Start:
<pre>
./tiktactou.out
</pre>
# How to play

If it's your turn, choose field with nummbers like so:

<pre>
 1  |  2  |  3 
---------------  
 4  |  5  |  6 
---------------  
 7  |  8  |  9
</pre>
 
good luck, bro.
